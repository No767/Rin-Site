Rin

A Discord bot with EasyBot.py plugin support and third party API support

collections

A wide collection of services supported

Rin supports a wide variety of services, allowing to obtain data from these services just from within Discord (For example, you can request some art from DeviantArt, or search up whether a Minecraft server is online or not). Such services include Hypixel, Instagram, and many more. More will be coming in the near future, indcluding Mangadex and YT. (The Mangadex service is also planned to allow you to read manga from Discord)

speed

Optimized for Performance

This bot is designed for speed, and does that without sacrificing functionality. This allows for more control over how the bot scales and performs. Which means that the bot is also going to be able to return the data pretty quickly as well.

Completely Open Source

Rin's source code remains completely free and open source, and it is licensed under Apache-2.0. This allows for a level of transparency, and thus allowing anyone to contribute to the bot if they wish so. In short, it's not dangerous to use.
