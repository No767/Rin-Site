# Building

Dependencies:

- [Node.js](https://nodejs.org/en/)
- Pnpm

## React

Installing Dependencies:

```sh
cd rin-next && pnpm install
```

Opening up the dev server:

```sh
pnpm run dev
```

Building Files for Production:

```sh
pnpm run build
```
